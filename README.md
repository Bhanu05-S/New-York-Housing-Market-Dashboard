# New-York-Housing-Market-Dashboard
Interactive Excel dashboard analyzing New York housing market trends using Pivot Tables, Pivot Charts, and slicers.
This dashboard transforms raw housing data into meaningful business insights using:
- Pivot Tables
- Pivot Charts
- Interactive Slicers
- KPI Cards
- Data Visualization Techniques
The project helps users explore property prices, locality trends, listing distribution, property sizes, and market segmentation through an intuitive and interactive reporting interface.

# 📊 Key KPIs

| KPI | Description |
|---|---|
| Total Properties | Total number of listings analyzed |
| Average Bedrooms | Average bedroom count |
| Average Property Sqft | Average property size |
| Average Property Price | Average housing price |

# ✨ Features

## 🔹 Interactive Filters

The dashboard includes slicers for dynamic filtering by:

- Property Type
- Bedrooms
- Locality

## 🔹 Dashboard Insights

### 📌 Average Price by Property Type
Compares pricing across different property categories.

### 📌 Average Price by Bedrooms
Analyzes how bedroom count impacts property prices.

### 📌 Listings by Locality
Displays listing concentration across New York regions.

### 📌 Property Size by Locality
Compares average property sizes geographically.

### 📌 Price Distribution
Segments properties into:
- Affordable
- Mid Range
- Luxury

### 📌 Average Price by Locality
Highlights high-value neighborhoods and pricing trends.
